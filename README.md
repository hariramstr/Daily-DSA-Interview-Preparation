## 📖 What Is This Repository?

This repository is updated daily with new coding problems covering the most common data structures and algorithms asked in technical interviews.
Each problem folder contains:

| File | Purpose |
|------|---------|
| `README.md` | Plain-English explanation + full technical walkthrough |
| `meta.json` | Machine-readable problem metadata (title, difficulty, tags) |
| `csharp/Program.cs` | C# solution with detailed comments |
| `python/solution.py` | Python solution with type hints and comments |
| `java/Solution.java` | Java solution with Javadoc comments |
